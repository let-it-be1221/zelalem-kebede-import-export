# Zelalem Kebede General Import & Export

> Official corporate website for **Zelalem Kebede General Import and Export** — one of Ethiopia's leading trading companies engaged in international import, export, manufacturing, and fuel distribution.

---

## 🌐 Live Preview

The website is a modern, responsive single-page application showcasing the company's full portfolio of services, global dealer network, facilities, and contact information.

---

## 🏢 About the Company

**Zelalem Kebede General Import and Export** was established in **2009 E.C.** under Ethiopian Commercial Law as a Sole Proprietorship. Led by **Mr. Zelalem Kebede**, the company has grown into a multi-vertical trading powerhouse with **300+ employees** and total assets exceeding **5 Billion ETB**.

### Core Business Verticals

| Vertical | Description |
|----------|-------------|
| **Export** | Premium Ethiopian agricultural products — oilseeds, cereals, spices, pulses, coffee, tea, and chat |
| **Import** | World-class vehicles, construction machinery, chemicals, and spare parts from global manufacturers |
| **Manufacturing** | Pasta & biscuit plant, corn & soya blending line, and sesame seed cleaning plant |
| **Fuel Distribution** | Fuel station operations, LPG, lubricants, and fuel transportation services |

### Key Stats

- **300+** permanent and contract employees across Ethiopia
- **5 Billion+ ETB** in total assets
- **42,000+ m²** of land at Bure Industry Park
- **5** offices and warehouse locations nationwide
- **Global dealer network** spanning China, Korea, Africa, Europe, and Middle Asia

## ✨ Website Features

### Sections

| Section | Highlights |
|---------|-----------|
| **Navigation** | Sticky navbar with transparent-to-solid scroll effect, mobile-responsive hamburger menu |
| **Hero** | Full-viewport banner with animated company name, tagline, and CTA buttons |
| **About** | Company overview, mission & vision cards, animated stat counters |
| **Services** | Four interactive service cards with images and hover effects |
| **Business Lines** | Tabbed detail view with comprehensive product listings and vehicle brands |
| **Global Reach** | Visual representation of international dealer network across 5 regions |
| **Facilities** | Infrastructure and asset showcase with key statistics |
| **Contact** | Contact form with toast notifications, phone numbers, address, and map placeholder |
| **Footer** | Multi-column layout with quick links, business lines, contact info, and back-to-top button |

### Design & UX

- **Color Scheme:** Dark Green (`#0D4B2A`) primary with Gold (`#C8A951`) accent — professional and premium
- **Animations:** Smooth scroll, fade-in-on-scroll effects, animated counters, and interactive hover states using Framer Motion
- **Responsive:** Fully mobile-first design — works flawlessly on phones, tablets, and desktops
- **Accessibility:** Semantic HTML5, ARIA attributes, keyboard navigation, and proper alt text
- **Performance:** Optimized images, lazy loading, and efficient rendering with Next.js

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) 18+ or [Bun](https://bun.sh/)
- Git

### Installation

```bash
# Clone the repository
git clone https://github.com/let-it-be1221/zelalem-kebede-import-export.git

# Navigate to the project directory
cd zelalem-kebede-import-export

# Install dependencies
bun install
# or
npm install
```

### Development

```bash
# Start the development server
bun run dev
# or
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the website.

### Production Build

```bash
# Build for production
bun run build
# or
npm run build

# Start the production server
bun run start
# or
npm run start
```

### Linting

```bash
# Run ESLint
bun run lint
# or
npm run lint
```
